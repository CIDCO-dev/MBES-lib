◆ detectedRangeInSamples

uint16_t detectedRangeInSamples

Two way range in samples. Approximation to calculated distance from tx to bottom detection [meters] = soundVelocity_mPerSec * detectedRangeInSamples / (sampleFreq_Hz * 2). The detected range is set to zero when the beam has no bottom detection. Replaced by detectedRangeInSamplesHighResolution for higher precision.

◆ sampleAmplitude05dB_p

int8_t* sampleAmplitude05dB_p

Pointer to start of array with Water Column data. Lenght of array = numSampleData. Sample amplitudes in 0.5 dB resolution. Size of array is numSampleData * int8_t. Amplitude array is followed by phase information if phaseFlag >0. Use (numSampleData * int8_t) to jump to next beam, or to start of phase info for this beam, if phase flag > 0.
